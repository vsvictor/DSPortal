# Портал ДП "Цифрове"

Flutter-портал для ДП "Цифрове" (Міністерство економіки України) з відкритою та закритою частинами.

## Що реалізовано

- Державна візуальна стилістика (синьо-жовта палітра, тематичний хедер/футер).
- Відкрита частина з окремими сторінками:
  - Про нас
  - Послуги
  - Проекти
  - Новини
  - Вакансії
  - Звернення
  - Контакти
- Закрита частина (`Кабінет`) доступна лише авторизованим користувачам.
- Реєстрація та вхід (демо-режим, локальна in-memory модель).
