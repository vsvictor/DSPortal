# Портал ДП "Цифрове"

Flutter-портал для ДП "Цифрове" (Міністерство економіки України) з відкритою та закритою частинами.

## Що реалізовано

- Державна візуальна стилістика (синьо-жовта палітра, тематичний хедер/футер).
- Відкрита частина з окремими сторінками:
  - Про нас
  - Послуги
  - Проекти
  - Новини
  - Вакансії
  - Звернення
  - Контакти
- Закрита частина (`Кабінет`) доступна лише авторизованим користувачам.
- Кросплатформна модальна автентифікація:
  - Зручні модальні вікна (діалоги) для входу та реєстрації.
  - Адаптивний UI (автоматичне налаштування відступів і шрифтів для мобільних пристроїв та десктопу).
  - Функціонал показу/приховування пароля та валідації полів (вкл. підтвердження пароля).
- Локальна in-memory модель для збереження стану авторизації.
